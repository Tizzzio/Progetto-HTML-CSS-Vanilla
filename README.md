## Titolo del Progetto

TizzzVerse - Il Museo del Videogioco

## Struttura del Progetto

- `index.html`: La pagina principale del sito.

- `index.css`: Il file CSS che contiene gli stili per il sito.

- `img/`: Cartella contenente le immagini utilizzate nel sito.

- `README.md`: Questo file di documentazione.

- `eventi.html`: Pagina dedicata agli eventi legati ai videogiochi.

- `galleria.html`: Pagina dedicata alla galleria di immagini.

- `esposizioni.html`: Pagina dedicata alle esposizioni di videogiochi.

- `console.html`: Pagina con le informazioni di contatto.

## Descrizione delle Pagine

- **Home Page**: Presenta una panoramica del sito con immagini e testi accattivanti.

- **Pagina Esposizioni**: Elenca gli eventi legati ai videogiochi con dettagli e immagini.

- **Pagina Consoles**: Mostra le console che hanno fatto la storia.

- **Pagina Galleria**: Mostra Immagini di videogiochi che hanno lasciato un'impronta duratura nella storia.
- **Pagina Eventi**: Visualizza quali sono i maggiori Eventi a livello globale a tema videogiochi

## Tecnologie Utilizzate

- HTML

- CSS

## Come Visualizzare il Progetto

1. Clona o scarica il repository sul tuo computer.

2. Apri il file `index.html` in un browser web per visualizzare il sito
